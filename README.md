# Breweries

### Run project in development

1. Clone the repo
   ```sh
   git clone https://github.com/iwangsyah/Breweries.git
   ```
2. Install dependencies
   ```sh
   npm install
   ```
   or
   
   ```sh
   yarn install
   ```
3. Start the app
  - Run on Android: `yarn android` (or `npm run android`).

  - Run on iOS: `yarn ios` (or `npm run ios`).
